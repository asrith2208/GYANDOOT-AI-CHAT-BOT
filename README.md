# ज्ञानदूत (Gyandoot) - AI Chatbot for Uttaranchal University

This repository contains the code for **Gyandoot**, an AI-powered chatbot personalized for Uttaranchal University, developed within Firebase Studio.

Gyandoot is designed to assist prospective students, parents, and staff by providing accurate, instant, and motivational information about the university. It leverages advanced AI to create a supportive and engaging user experience.

### Key Features:

*   **Multilingual Conversations:** Answers queries in multiple Indian languages, including English, Hindi, Telugu, and more, complete with regional dialect understanding.
*   **Voice Interaction:** Includes both **Text-to-Speech** and **Speech-to-Text** capabilities for a fully accessible, hands-free communication experience.
*   **Motivational AI:** The chatbot is programmed to be encouraging and positive, framing information to highlight the university's strengths and inspire potential students.
*   **Official Knowledge Base:** Gathers all its information from Uttaranchal University's official websites, ensuring every answer is accurate and up-to-date.

### Tech Stack:

*   **Frontend:** Next.js, React, Tailwind CSS
*   **AI Backend:** Google's Genkit framework with Gemini models
*   **UI Components:** ShadCN
