# ज्ञानदूत (Gyandoot) - AI Chatbot for Uttaranchal University

Hey all! I've developed an AI chatbot, **Gyandoot**, personalized for Uttaranchal University.

This isn't just a standard bot. Gyandoot is designed to be a supportive and engaging assistant for students, parents, and university staff.

### Key Features:

*   **Multilingual Conversations:** It can answer queries in multiple languages, including English, Hindi, Telugu, and much more, complete with regional dialect understanding.
*   **Voice Interaction:** Includes both **Text-to-Speech** and **Speech-to-Text** capabilities for a fully accessible, hands-free communication experience.
*   **Official Knowledge Base:** Gathers all its information from Uttaranchal University's official websites, ensuring every answer is accurate and up-to-date.
*   **Motivational AI:** The chatbot is programmed to be encouraging and positive, helping to create a supportive environment for prospective students.

### Tech Stack:

*   **Frontend:** Next.js, React, Tailwind CSS
*   **AI Backend:** Google's Genkit framework with Gemini models
*   **UI Components:** ShadCN
